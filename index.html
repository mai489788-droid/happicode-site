Hướng dẫn sử dụng

Làm theo từng bước để bắt đầu với mã nguồn của bạn

1

Cài đặt Visual Studio Code

Để bắt đầu, bạn cần tải và cài đặt Visual Studio Code (VS Code) trên máy tính của mình. Truy cập trang web chính thức của VS Code tại https://code.visualstudio.com/ và chọn phiên bản phù hợp với hệ điều hành của bạn (Windows, macOS hoặc Linux). Sau khi tải xuống, hãy chạy tệp cài đặt và làm theo hướng dẫn để hoàn tất quá trình cài đặt.

2

Cài đặt Extension

Mở Visual Studio Code và truy cập vào phần Extensions bằng cách nhấp vào biểu tượng Extensions ở thanh bên trái hoặc sử dụng phím tắt Ctrl + Shift + X (Windows/Linux) hoặc Cmd + Shift + X (macOS). Trong thanh tìm kiếm, bạn nhập "Live Server", phiên bản phổ biến nhất được phát triển bởi Ritwick Dey. Khi bạn tìm thấy nó, nhấp vào nút Install để cài đặt nó vào VS Code.

3

Lấy Source Code

Bạn truy cập vào mục Code hoặc trang Github tìm kiếm mã nguồn mà bạn muốn để tải về.
Sau khi tải về, Source Code thường sẽ nằm ở mục Download của máy bạn. Bạn giải nén và mở thư mục dự án trong Visual Studio Code.

Lưu ý: Chỉ làm bước này nếu như bạn chưa có Source Code. Nếu bạn đã được mình gửi Source Code thì chỉ cần giải nén và mở ra để sử dụng.

4

Mở Source Code

Sau khi giải nén, bạn sẽ được một thư mục chứa các tệp như "index.html", "style.css", "script.js",...và nhiều file khác tuỳ theo Source Code.
Bạn mở Visual Studio Code, chọn "File" -> "Open Folder", bạn đi tới mục lưu Source Code đã giải nén, chọn thư mục đó và chọn "Select Folder".

5

Chạy Code trong VSC

Sau khi mở được Source Code, bạn sẽ thấy được cấu trúc thư mục như hình ảnh.
Bạn có thể chỉnh sửa bất kỳ thông tin như lời nhắn, hình ảnh, lời chúc...
Để chạy chương trình, bạn vào index.html và có 2 cách khởi chạy:
1. Chọn vào biểu tượng bên góc phải dưới màn hình.
2. Chuột phải chọn vào "Open with Live Server".
